## <input>
- <input> 유형은 사용자로부터 다양한 유형의 입력을 받을 수 있도록 한다.
### <input>의 type
- **text** : 일반적인 텍스트 입력 필드
- **number** : 숫자 입력 필드
- **email** : 이메일 주소 입력 필드

## forEach()
- `forEach()`는 배열의 각 요소에 대해 주어진 함수를 실행한다.
- `array.forEach(function(element, index, array)`
- **element** : 배열의 현재 요소
- **index** : 현재 요소의 인덱스
- **array** : `forEach`를 호출한 배열 자체의 참조
